# Posto Via 14

Projeto React + Vite corrigido.

## Como rodar localmente

```bash
npm install
npm start
```

Acesse: http://localhost:3000

## Build de produção

```bash
npm run build
```

Os arquivos finais ficam na pasta `dist/`.

## Login demo do painel

- Usuário: `admin`
- Senha: `123`

Também aceita o usuário `admin@via14.com.br` com a mesma senha.
